## pks delete-cluster

Deletes a kubernetes cluster, requires cluster name

### Synopsis

Delete-cluster requires a cluster name.

```
pks delete-cluster <cluster-name> [flags]
```

### Examples

```
  pks delete-cluster my-cluster
```

### Options

```
  -h, --help   help for delete-cluster
```
