## pks plans

View the preconfigured plans available

### Synopsis

This command describes the preconfigured plans available 

```
pks plans [flags]
```

### Examples

```
  pks plans
```

### Options

```
  -h, --help   help for plans
      --json   Return the PKS-API output as json
```
