## pks clusters

Show all clusters created with PKS

### Synopsis

This command describes the clusters created via PKS, and the last action taken on the cluster

```
pks clusters [flags]
```

### Examples

```
  pks clusters
```

### Options

```
  -h, --help   help for clusters
      --json   Return the PKS-API output as json
```
